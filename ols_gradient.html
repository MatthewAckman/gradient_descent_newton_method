OLS vs gradient descent¶

Gradient descent moves parematers from their initial guess (0,0) towards the minimum of a surface defined by the cost function. The cost function in this case is defined by the partial derivatives of the SSE (or MSE? Does it matter?) wrt b0 and b1.

$$SSE = \Sigma{(y_{i} - \hat{y})^2} = \Sigma{(y_{i} - \beta_{0} - \beta_{1}x_{i})^2}$$$$\dfrac{\partial{SSE}}{\partial{\beta_{0}}} = -2\Sigma{(y_{i} - \beta_{0} - \beta_{1}x_{i})} = -2\Sigma{x_{i}y_{i}}$$$$\dfrac{\partial{SSE}}{\partial{\beta_{1}}} = -2x_{1}\Sigma{(y_{i} - \beta_{0} - \beta_{1}x_{i})} = -2\Sigma{y_{i}}$$

Start $\beta_{0}$ and $\beta_{1}$ at 0 and, with each epoch, subtract the value of the 2nd derivative multiplied by the learning rate $\alpha$

$$\beta_{0} = \beta_{0} - \alpha \dfrac{\partial{SSE}}{\partial{\beta_{0}}}$$$$\beta_{1} = \beta_{1} - \alpha \dfrac{\partial{SSE}}{\partial{\beta_{1}}}$$

Where $\alpha$ is the size of the step taken with each epoch (iteration), larger steps find the solution more quickly but risk overshooting the optimum. As the optimum is approached, the partial derivatives approach 0 and the estimated for b0 and b1 approach thier optimum values asymptotically.

## Define functions for gradient descent

def MSE(y,x,b0,b1):
    '''
    yields mean squared error of linear model given ols params

    MSE = (y_{i} - b0 - b1*x_{i})**2 * n^-1
    '''
    assert len(x)==len(y)

    tot_err_vec = (y - b0 - b1 * x)**2

    return tot_err_vec.sum() / len(y)


def gd(y,x,b0=0,b1=0,alpha=0.01,n_epochs=15001):
    '''
    inclusive gradient decent function

    y = endog var
    x = exog var
    b0,1 = 0  <--  initial guess for b0,1
    alpha = 0.01  <-- learning rate
    n_epochs = 15001  <-- number of iterations
    '''
    results=pd.DataFrame(columns=['b0','db0','b1','db1','mse'])  # make empty dataframe to store results every 1000th iter

    for epoch in range(n_epochs):

        # yield partial derivatives
        dmse_db0 = (-2/len(y)) * (y - (b0 + b1*x)).sum()  # der of MSE wrt beta_{0}
        dmse_db1 = (-2/len(y)) * (y * (y - (b0 + b1*x))).sum()  # der of MSE wrt beta_{1}

        # update params
        b0 -= alpha * dmse_db0  # b0 = b0 - a * dMSE/dB0
        b1 -= alpha * dmse_db1

        if epoch % 1000 == 0:
            results.loc[epoch] = (b0, dmse_db0, b1, dmse_db1, MSE(y=y,x=x,b0=b0,b1=b1))
        
    return results
